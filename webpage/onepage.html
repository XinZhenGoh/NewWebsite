What is a singleton class?

A class that can only be instantiated once

Explain inheritance

A class can inherit the implementation of its parent class, while also extending/ changing parts of it. Allows code reuse when things have in common

Describe a deadlock situation

Thread A without releasing x requests y, but wait for B to release it. Thread B, without releasing y, requests resource x,but have to wait for A to release it. Both threads are now waiting for each other to release their resource, but none will until the other does.

What is a non functional requirement?

It specifies general behavior requirements of the system. It describes what you want your system to be, rather then what it should do. accessibility, maintainability & security.

What is functional requirement?

A functional requirement describes the specific functionality you want of your system.

What is modularization?

Modularization is a technique where the software is divided into several modules that work independently of each other. Better maintainability.

What is the MVC architecture?

Used most commonly for GUI applications, seperating data from user interface. Model layer holds the data, the view layer outputs the data to user and controller is responsible for making changes to the model based on user input.

What is a directed acyclic graph?

A directed (edges have direction) graph without any cycles. It is impossible to start at a node, and follow and directed edges return to that same node).

What is the difference between black box and white box testing?

Black box testing only tests a function for correct output given an input, white box testing also tests if the implementation is correct.

What is the difference between a queue and a stack?

A queue is based on the FIFO - First In, First Out - principle, first to enter, first to be removed.
A stack is based on the LIFO - Last In, First Out - last to enter, first to go out.

What is a recursive function?

A recursive function is a function that calls itself. At each function call, one or more parameters are changed. The recursion continues until a base case (a set of parameters which return a value) is reached, and the recursion is resolved.

What is the average and worst case run time of the quicksort algorithm?

How does the A* algorithm work?

A* is a path finding algorithm that words on weighted graphs with a heuristic function that estimates the cost of travelling from node n to the goal node.
The nodes to be considered at each step is stored in a collection/priority queue. It starts by adding every node that is adjacent to the start node into the queue. Using the formula: f(n) = g(n) + h(n).
g(n) is the actual cost from the start node to node n.
h(n) is the heuristic function.

What is inner and outer join?

Inner join joins the intersection between the tables, only keys that exist in both tables will be kept in joined table.
Outer join works as the union between the tables, all rows will be included in the joined table. (Left outer join keeps all rows from the left table, while only the rows with keys that also exists in the left table are included from the right table.

What is an SQL injection, and how they are prevented?

SQL injection is when SQL statements are entered into data fields that are to be stored in an SQL database. The most usual mitigation techniques include parameterized statements, escaping characters with meaning in SQL, pattern checking and limiting database permissions.

What is overloading and overriding?

Overloading occurs when two or more methods in one class have the same method name but different parameters. Overriding means having two methods with the same method name and parameters
